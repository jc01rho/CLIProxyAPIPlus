# 발견된 문제점들

## trae2api 프로젝트 관련 이슈

### 1. 프로젝트 상태
- **아카이브됨**: 2025년 8월 11일부터 읽기 전용
- **업데이트 중단**: 더 이상 새로운 이슈나 PR 받지 않음
- **버전 제한**: Trae v1.3.0 이후 모델 미지원

### 2. "param is invalid" 오류
**원인:**
- API 엔드포인트 변경
- 필수 헤더 누락
- 토큰 형식 변경
- 환경변수 설정 오류

**해결 방법:**
- 최신 API 엔드포인트 사용
- 모든 필수 환경변수 설정 확인
- 토큰 갱신 프로세스 재구현

### 3. 네트워크 캡처 어려움
**Electron 앱 특성:**
- Main/Renderer 프로세스 분리
- 기본 DevTools로 모든 요청 캡처 불가
- HTTPS 트래픽 암호화

**해결책:**
- HTTP Toolkit 사용 (Electron 전용 지원)
- mitmproxy + 환경변수 설정
- Wireshark + TLS 키 로그

## Trae IDE 자체 이슈

### 1. 인증 문제
- "Failed to get login host" 오류
- Pro 계정 기능 제한
- OAuth 토큰 만료 처리

### 2. 네트워크 제한
- CORS 정책 차단
- 프록시 환경에서 AI-Complete 기능 불가
- 요청 제한 및 대기열

### 3. 지역별 제한
- 국제버전 결제 불가
- API 엔드포인트 지역별 차이
- 일부 기능 지역 제한
